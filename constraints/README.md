# FPGA Constraints

This folder will contain the FPGA board constraint file used for hardware implementation.

The exact `.xdc` file will be added after confirming the FPGA development board available in the school lab.

## Planned Signal Mapping

| Signal | FPGA Hardware |
|---|---|
| clk | On-board clock |
| reset | Push button |
| ped_button | Push button |
| emergency | Switch |
| main_red | LED |
| main_yellow | LED |
| main_green | LED |
| side_red | LED |
| side_yellow | LED |
| side_green | LED |
| ped_walk | LED |
